## 控制参数说明

| 参数名称 | 参数说明 |
|---------|---------|
| height | 雪碧图小图的高度，单位：像素 |
| width |  雪碧图小图的宽度，单位：像素 |
| row | 单张雪碧图最多包含多少行小图 |
| colunm | 单张雪碧图最多包含多少列小图 |
| interval | 每张小图的截图时间间隔 |
| fillType | 如果视频分辨率与雪碧图小图分辨率不一致，采取何种填充方式；备选方式：拉伸、留黑、留透明 |
