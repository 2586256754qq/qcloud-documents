## 接口名称
DeleteClass

## 功能说明
1. 删除视频分类。

## 请求方式

### 请求域名
vod.api.qcloud.com

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| classId | 是 | int | 待删除视频分类的ID。 |

### 请求示例
```
https://domain/v2/index.php?Action=DeleteClass
&classId=108
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Int | 错误码, 0: 成功, 其他值: 失败 |
| message | String | 错误信息 |

### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见公共错误码。  |
| 1000 | 无效参数  |
| 1904 | 父分类不存在 |
| 1905 | 分类没有找到 |
| 2000 | 访问数据库失败  |

### 应答示例
```
{
    "code": 0,
    "message": ""
}
```
