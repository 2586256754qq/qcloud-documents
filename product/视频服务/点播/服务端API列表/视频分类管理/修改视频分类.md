## 接口名称
ModifyClass

## 功能说明
1. 修改视频分类的属性，包括名称。

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| classId | 是 | Int | 待修改的分类id |
| className | 是 | String | 新的分类名 |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=ModifyClass
&classId=100
&className=test
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Int | 错误码, 0: 成功, 其他值: 失败 |
| message | String | 错误信息 |

### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见公共错误码。  |
| 1 | 访问数据库出错  |
| 1000 | 无效参数  |
| 1021 | 数据库更新分类失败 |
| 1905 | 分类没有找到 |
| 1907 | 数据库存在重复分类名 |

### 应答示例
```javascript
{
    "code": 0,
    "message": ""
}
```
