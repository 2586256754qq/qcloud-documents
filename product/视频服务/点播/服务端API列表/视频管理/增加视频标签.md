## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| fileId | 是 | String | 视频的ID |
| tags.n | 是 | String | 标签列表，字母数字和汉字，单个标签限长8个字 |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=CreateVodTags
&fileId=2721945854681023354
&tags.1=testTag1
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Integer | 错误码, 0: 成功, 其他值: 失败 |
| message | String | 错误信息 |

### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见公共错误码  |
| 1 | 访问数据库出错  |
| 1000 | 无效参数  |
| 1013 | 数据库添加标签失败  |
| 10008 | 文件不存在  |

### 应答示例

```javascript
{
    "code": 0,
    "message": ""
}
```
